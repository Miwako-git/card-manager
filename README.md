# 🗂 Card Manager App

[日本語](#japanese) | English

A mobile-first web app for managing cards — hospital cards, ID cards, membership cards, and more — with photos.  
Built with AI collaboration (vibe coding).

## ✨ Features

- 📸 **Camera capture** — Real-time camera with horizontal guide line
- ✂️ **Crop tool** — Drag to move, pinch to zoom
- 🗂 **Card list** — Category filter & keyword search
- ✏️ **Edit** — Update card name & category anytime
- 📤 **Backup** — Export / import all data including photos as JSON
- 📱 **PWA support** — Add to home screen and use as an app

## 🛠 Tech Stack

- HTML / CSS / JavaScript (no frameworks)
- IndexedDB (browser-side database)
- Web Camera API

## 🚀 Usage

1. Visit [my-card-box.netlify.app](https://my-card-box.netlify.app)
2. On iPhone: Open in Safari → Share → Add to Home Screen
3. Register your cards and use it

## 💾 About Data

All data is stored locally in your browser's IndexedDB.  
No data is sent to any server.  
When switching devices, export from the Backup tab and import on your new device.

## 📝 Notes

- Developed through AI collaboration (vibe coding)
- Code is heavily commented — useful as a learning reference

# 🗂 カード管理アプリ

病院の診察券・身分証・ポイントカードなどを写真付きで管理するスマホ向けWebアプリです。  
AIとの協働（バイブコーディング）で開発しました。

## ✨ 機能

- 📸 **カメラ撮影** — リアルタイムカメラ＋水平ガイドライン付き
- ✂️ **トリミング** — ドラッグ移動・ピンチで拡大縮小
- 🗂 **カード一覧** — カテゴリフィルター・キーワード検索
- ✏️ **編集機能** — カード名・カテゴリをあとから変更可能
- 📤 **バックアップ** — 写真データごとJSONエクスポート／インポート
- 📱 **PWA対応** — ホーム画面に追加してアプリとして使える

## 🛠 使用技術

- HTML / CSS / JavaScript（ライブラリなし）
- IndexedDB（ブラウザ内データベース）
- Web Camera API

## 🚀 使い方

1. [my-card-box.netlify.app](https://my-card-box.netlify.app) にアクセス
2. iPhoneの場合：Safari で開いて「共有」→「ホーム画面に追加」
3. カードを登録して使う

## 💾 データについて

データはすべてブラウザの IndexedDB に保存されます。  
外部サーバーへの送信は一切ありません。  
機種変更のときは「バックアップ」タブからエクスポートして、新しい端末でインポートしてください。

## 📝 開発メモ

- AIとの協働（バイブコーディング）で開発
- コード内にコメントをたっぷり入れてあるので、学習用としても参考になります
